# Beechland Farms

A simple web app for managing farms, fields, and image uploads.

## Features
- Home page with quick navigation
- Farms page to add and list farm entries
- Farm details page showing fields and image upload support
- Data is stored locally in your browser using `localStorage`

## How to use
1. Open `index.html` in a browser.
2. Add farms on the Farms page.
3. Click a farm to see its fields page.
4. Add fields and upload images for the selected farm.
